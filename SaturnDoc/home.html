Saturn Programming Language


Saturn is a dynamic scripting language. It follows a semi object orientated structure. It supports classes and inheritence, however, base types like numbers and strings are not classes, hence semi-object orientated. The official interpreter is currently using a C# written interpreter, however, it will be moving to a C bytecode interpreter. Once this interpreter is made, it will be released as open source.

Saturn is based on a mix of languages. The most common of those is Javascript, Python and C. It used the C syntax so curly braces are used in code blocks and for loops work like C's and not Pythons for Range. It also uses some elements from Javascript such as the let keyword, and function. Unlike Javascript however, Saturn requires semicolons on the end of its statements, other a syntax error will be throw.
Saturn also borrows one element from C#, as it has two different functions to print to the console. One will put a new line character on the end, and the other will not, similar to C#'s Write and WriteLine methods. 



function Fib(n)
{
    let n1 = 0;
    let n2 = 1;
    let count = 0;

    if (n <= 0)
    {
        Println("Please enter a positive integer");
    }
    else if (n == 1)
    {
        Println("Fibonacci sequence to " + n);
        Print(n1 + " ");
    }
    else
    {
        Println("Fibonacci sequence");
        while (count < n)
        {
            Print(n1 + " ");
            let nth = n1 + n2;
            n1 = n2;
            n2 = nth;
            count = count + 1;
        }
    }
}

function Init()
{
    SetCursorVisibility(false);
    Clear();
}

Init();
Fib(StrToNum(Input("Please enter the number of digits to calculate")));
