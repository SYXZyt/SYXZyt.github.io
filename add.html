Add command

The add command will add up all of the values that have been stored in the accumulator. If the accumulator is empty, an error will be thrown. A different result will occur, depending on the values stored in the accumulator. If any of the values stored are the type of string, then the addition will be treat like a string concat.

lda "d";
lda 1;
add;
The code above will result in "1d" being pushed to the stack. If only ints or floats are in the accumulator, then it will be treat like a numeric addition. This is the only accumulator command that will accept strings.
This example will show numeric addition.
lda 1;
lda 9;
add; //This will push 10 to the stack
println pop; //Print the result to the screen
