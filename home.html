Titan

Titan is a stack based programming language. It is still under active development currently in version 1.0.
The language features many advances features such as conditional programming; such as if and else statements. The language can support user written functions, and libraries in a future build. The interpreter was written in C# and has an estimate 2k lines powering it, although every update drastically changes that. Functions are also to play a larger part of the language in a future build.


Example program
ldi 0;
while ind lwr 10;
{
//Push the current indexer value, to the stack
push ind;
push "indexer: ";

//Draw a message along with the indexer
print pop;
println pop;

//Increment the index, to avoid an infi loop
inc;
};
