My Profiles

SPL

SPL is a simple scripting language, which features x86 and BASIC inspired syntax. The language was created as a learning tool into interpreter design. The language features a custom-made bytecode platform, which the language compiles to.



Stranded In Space

Stranded In Space was the first game I finished and released. It came out in 2020 and was made during my time at college. The game is a 2D platformer where you must beat levels as fast as you can.

The game used GameJolt integration to support achievements.



Stranded In Space II

Immediately after the release of Stranded In Space a sequel began development. This game's aim was to improve on from the first, and to allow me to use new skills I had developed over the course of Stranded In Space's development.

Stranded In Space 2 natively supports 1080p which allows for larger levels, without having to deal with camera movement.
An official soundtrack was produced specifically for use in Stranded In Space 2.



Astral Scripting Language

Astral is a lightweight scripting language, designed to be easy to integrate. It can interop with C++ for additional functionality
It is a C-styled language and features no objects and so structs and references can be used instead.
The language was primarily created for use in my game engine project.
Astral is currently in development and not all features are currently implemented.
